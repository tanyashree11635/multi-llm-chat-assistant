# Multi LLM Chat Assistant

A modern chatbot application supporting OpenAI GPT and Google Gemini models with a clean Streamlit interface.

## Features
- OpenAI + Gemini model switching
- Real-time chat interface
- Session management
- Export chat history
- Secure API key handling
- Streamlit UI

## Tech Stack
Python, Streamlit, OpenAI API, Gemini API

## Run Locally
pip install -r requirements.txt
streamlit run frontend/streamlit_app.py
